# **CON TRỎ**
## 1. ĐỊNH NGHĨA
- Một con trỏ là một biến mà trong đó giá trị của nó là địa chỉ của biến khác.
## 2. ĐẶC ĐIỂM
### a,phân loại
- Con trỏ kiểu int dùng để chứa địa chỉ các biến kiểu int. Tương tự ta có con trỏ kiểu float,kiểu double vv.
